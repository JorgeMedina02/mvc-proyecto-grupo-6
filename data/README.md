# Instrucciones de descarga del Dataset

Para replicar este proyecto, descarga los datos oficiales del INE:
1. Accede a la Encuesta de Inserción Laboral de Universitarios (EILU 2019).
2. Descarga el fichero `EILU_GRAD_2019.csv` y el diccionario `dr_EILU_GRAD_2019.xlsx`.
3. Colócalos directamente en esta carpeta (`data/`).
